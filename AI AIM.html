AI AIM

AI นี้ใช้การพูดและการจับเสียงเป็นตัวจำแนกข้อความจึงสามารถสื่อสารผ่านเสียงได้ ให้ความสะดวกในการดำเนินการต่างๆ ผ่านคำสั่งของเสียง AI โปรแกรมที่ใช้การรับและประมวลผลจากเสียง และมีฟังก์ชั่นต่างๆ เพื่อดำเนินการตามคำสั่งของผู้ใช้นอกจากนี้ยังใช้ API ต่าง ๆ เช่น Wikipedia API, YouTube API, Google Search API, และ OpenWeatherMap API เพื่อดึงข้อมูลต่าง ๆ ตามที่ผู้ใช้ต้องการ เเละยังมี AI ที่น่าสนใจ คือ Speech Recognition และ Text-to-Speech (TTS) โดยใช้ไลบรารี SpeechRecognition และ pyttsx3 ตามลำดับ
